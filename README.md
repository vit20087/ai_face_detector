# 🧑‍💻 Інтелектуальна система розпізнавання облич (Face Detection App)

Цей проєкт — це веб-додаток для автоматичного розпізнавання облич на фотографіях, розроблений у рамках комплексної розрахункової роботи з курсу **«Системи штучного інтелекту»**. В основі системи лежить алгоритм комп'ютерного зору (Каскади Хаара) та бібліотека OpenCV.

## 🌟 Основні можливості

- **Завантаження зображень**: Підтримка форматів `.jpg`, `.jpeg`, `.png`.
- **Робота в реальному часі**: Миттєва обробка зображення та виведення результату.
- **Гнучкі налаштування алгоритму**: 
  - `scaleFactor` (Масштаб) — налаштування чутливості до розміру облич.
  - `minNeighbors` (Чутливість) — відсіювання хибних спрацювань (наприклад, розпізнавання круглих предметів з тінями як обличчя).
- **Кастомізація**: Можливість змінювати колір обмежувальних рамок (bounding boxes) для кращого контрасту.

## 🛠 Технології

- **[Python](https://www.python.org/)** — основна мова програмування.
- **[Streamlit](https://streamlit.io/)** — фреймворк для створення зручного та інтерактивного веб-інтерфейсу без використання HTML/CSS.
- **[OpenCV](https://opencv.org/)** — математичне ядро для обробки зображень та машинного зору (метод `detectMultiScale`).
- **[NumPy](https://numpy.org/)** та **[Pillow (PIL)](https://pillow.readthedocs.io/)** — для роботи з масивами пікселів та форматами зображень.

## 🧠 Особливості алгоритму (Haar Cascades)
- **Алгоритм використовує метод Віоли-Джонса та ознаки Хаара.**
- **Він шукає специфічні перепади світла й тіні (наприклад, перенісся зазвичай світліше за очі).**
- **Якщо алгоритм помилково розпізнає інші предмети (наприклад, м'яч) як обличчя через специфічні тіні, достатньо збільшити параметр "Чутливість (minNeighbors)" у бічному меню програми.**
